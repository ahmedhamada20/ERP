# CoreX ERP — نظام إدارة شركات السياحة والسفر

## نظام عربي متكامل مبني خصيصاً لشركات السياحة الدينية والداخلية والدولية

---

## الفكرة في سطر

**CoreX ERP** هو نظام إدارة موارد مؤسسي (Enterprise Resource Planning) مبني من الصفر بواجهة عربية كاملة (RTL) ليغطي كل عمليات شركتك السياحية في منصة واحدة — من أول الـ Lead إلى تسليم العميل وإصدار العقد والإيصال، مروراً بالمحاسبة الكاملة والمرتبات والموردين.

النظام **ليس قالب جاهز معرّب**، بل بناء أصيل صُمم لقواعد عمل سوق السياحة الدينية والمصرية تحديداً.

---

## لماذا CoreX ERP؟

### 1. عربي بالكامل — لا ترجمة ولا تعريب
- واجهة RTL مصممة من الأساس بالعربية.
- طباعة العقود والإيصالات بخط Cairo باحتراف (Arabic shaping + BiDi).
- المراسلات والـ WhatsApp والإشعارات بالعربية.

### 2. سياحة دينية + داخلية في منصة واحدة
- إدارة حجوزات الحج والعمرة بكل تفاصيلها (المعتمرون، التأشيرات، السكن، الطيران، النقل، المطوّفون).
- إدارة الرحلات الداخلية والدولية بنفس الكفاءة.
- ربط حجوزاتك مباشرة بـ **منصة سفر (المسار)** ومنصة **اعتماد العمرة** عبر integration layer جاهز.

### 3. محاسبة كاملة مدمجة (مش محاسبة منفصلة)
- دليل حسابات (Chart of Accounts) مصمم لشركات السياحة المصرية.
- قيود يومية + سندات قبض + سندات صرف + ميزان مراجعة + قائمة الأرباح والخسائر + الأستاذ العام.
- **القيود المحاسبية بتنترل تلقائياً** من كل دفعة عميل، تكلفة مورد، فاتورة، أو مرتب — مش محتاج محاسب يكتب القيود يدوياً.

### 4. اعتمد على نظام محتوي 56 جدول، 44 موديل، 310 مسار
- **379 اختبار آلي** (Automated Tests) بيتأكدوا إن النظام شغّال بعد كل تعديل.
- معماري على Laravel 11 و MySQL يتحمل **+1 مليون سجل و+1000 مستخدم متزامن**.
- مفاتيح ULID (مش أرقام تسلسلية مكشوفة)، فهارس FULLTEXT للبحث السريع بالعربي.

## مكونات النظام بالتفصيل

### 1. وحدة السياحة الدينية (Religious Tourism)

> القلب النابض للنظام — 9 جداول و13 controller و43 route و120+ اختبار.

| المميزة | الوصف |
|---|---|
| إدارة الحجوزات | حج / عمرة بكل أنواعها (VIP، اقتصادي، شعبي) — حالات: مسودة، قيد المراجعة، مؤكد، جاري التنفيذ، مكتمل، ملغي |
| قائمة المعتمرين | كل معتمر بصورته، رقمه القومي، جوازه، صلاحيته، حالة التأشيرة، الباركود |
| إدارة التأشيرات | متابعة كل تأشيرة (قيد التقديم، صادرة، مرفوضة) + ربط آلي مع موقع وزارة الحج السعودية |
| السكن | حجز فنادق مكة والمدينة (يتربط مع ماستر الفنادق + المورد) |
| الطيران | تذاكر طيران مرتبطة بشركات الطيران (PNR، رقم الرحلة، المقاعد) |
| النقل | باصات / VIP / مطوّفين مرتبطة بمزودي النقل |
| الدفعات | متابعة كل دفعة من العميل (نقدي/بنكي) + إيصال PDF احترافي |
| التكاليف | تكلفة كل بند مرتبطة بالمورد المسؤول (فندق X، شركة طيران Y) → ربحية تلقائية لكل حجز |
| العقود | عقد PDF احترافي ينطبع تلقائياً بكل بنود الرحلة والمعتمرين |
| التنبيهات الذكية | تنبيه عند نقص ربحية الحجز، تأخر دفعة، عدم إصدار تأشيرة، اقتراب موعد سفر |
| تقارير الرحلات | قائمة المعتمرين (manifest)، تقرير الرحلة الكامل، تصدير Excel/PDF |

### 2. وحدة السياحة الداخلية والدولية (Domestic & International)

> 29 route، ربط محاسبي تلقائي، تقرير ربحية لكل برنامج.

- إدارة برامج سفر (Egypt، Dubai، Turkey، أي وجهة).
- نفس بنية الحجوزات: المسافرون، التكاليف، الدفعات، حالة الرحلة.
- تقرير **ربحية البرنامج (P&L by Program)** — يعرض لك أي برنامج هو الأعلى ربحية.

### 3. وحدة العملاء + إدارة العلاقات (CRM)

> 44 اختبار، 6 إشعارات WhatsApp تلقائية، مسارات تحويل (convert flows) كاملة.

- **Leads**: التقاط العملاء المحتملين من أي مصدر (Facebook، Google، واتساب، توصية).
- **Opportunities**: تحويل الـ Lead لفرصة مع تقدير قيمة وتاريخ متوقع للإغلاق.
- **تحويل تلقائي**: من Lead → Customer → حجز فعلي — مع الاحتفاظ بتاريخ الـ Lead.
- **WhatsApp Cloud API**: 6 رسائل تلقائية: تأكيد الحجز، تذكير بالدفع، إصدار التأشيرة، تذكير بالسفر، شكر بعد العودة، تحصيل المستحقات.
- نشاطات (calls، notes، follow-ups) بـ timeline لكل عميل.

### 4. وحدة المحاسبة الكاملة (Accounting)

> 120+ اختبار، Double-Entry Bookkeeping كامل، تكامل GL مع كل الموديولات.

| العنصر | الوصف |
|---|---|
| دليل الحسابات (Chart of Accounts) | شجري متعدد المستويات، مصمم لشركات السياحة المصرية، الكود يتولّد تلقائياً |
| قيود يومية (Journal Entries) | يدوية + مولّدة آلياً من النظام (لكل دفعة، تكلفة، فاتورة، مرتب) |
| سند قبض (Receipt Voucher) | استلام نقدية من عميل أو مورد |
| سند صرف (Payment Voucher) | صرف نقدية لمورد أو موظف |
| الخزائن والبنوك (Cash & Banks) | إدارة كل خزينة / حساب بنكي مع رصيد فعلي محسوب من القيود |
| ميزان المراجعة (Trial Balance) | أي تاريخ + تصدير Excel/PDF |
| الأستاذ العام (General Ledger) | حركة كل حساب بالتفصيل |
| قائمة الأرباح والخسائر (P&L) | عن أي فترة + مقارنات |
| فروقات العملة (FX) | جنيه/دولار/ريال مع أسعار صرف تاريخية |

### 5. وحدة الموردين والمدفوعات (Suppliers / AP)

> 78 اختبار، AP Subsidiary Ledger، FIFO Aging.

- إدارة موردي الفنادق، الطيران، النقل، التأشيرات.
- **فواتير الموردين** (Supplier Invoices) — تنترل قيد محاسبي تلقائي.
- **سندات صرف للموردين** — تنزل من فاتورتهم بترتيب FIFO.
- **كشف حساب المورد** (Statement) — كل العمليات بترتيب التاريخ.
- **تقرير أعمار الديون** (Aging Report) — 0-30، 31-60، 61-90، +90 يوم.

### 6. وحدة الموارد البشرية والمرتبات (HR + Payroll)

> 78+ اختبار، تأمينات + ضرائب + عمولات، GL Posting، Multi-Branch.

| المكوّن | الوصف |
|---|---|
| الفروع (Branches) | كل فرع له حساباته وموظفيه ومرتباته |
| الأقسام (Departments) | تنظيم هيكلي مع reports_to |
| الوظائف (Positions) | كل وظيفة مع حد أعلى / حد أدنى للراتب |
| الموظفون (Employees) | بيانات شاملة + مستندات + صور + حسابات بنكية |
| تشغيل المرتبات (Payroll Runs) | يحسب الراتب + التأمينات + الضرائب + العمولات + بدلات السكن والمواصلات + الخصومات |
| كشف راتب (Payslip) | PDF لكل موظف |
| السلف والقروض (Loans) | متابعة + خصم تلقائي من المرتب |
| القيد المحاسبي | كل دورة مرتبات تنزل قيد متوازن في الـ GL |

### 7. وحدة المخازن الذكية للمعرّفات (Master Data)

- ماستر الفنادق (مع التصنيف والمدينة).
- ماستر شركات الطيران.
- ماستر مزودي النقل.
- ماستر برامج السفر.
- ماستر أنواع التأشيرات.
- كل منها مرتبط بمورد محاسبي → تكلفة بنود الحجز تربط تلقائياً.

### 8. لوحة التحكم الذكية (Smart Dashboard)

- 6 بطاقات KPI رئيسية (أرباح، عملاء، تذاكر، حجوزات، تأشيرات، إيرادات الشهر).
- شريط انتقالات سريعة (Quick Actions) — حجز جديد، عميل، سند قبض/صرف، قيد محاسبي، تقارير.
- شريط نبض الأنظمة (Module Pulse) — حالة كل موديول في بطاقة واحدة.
- مخطط اتجاه الحجوزات والإيرادات (آخر 12 شهر).
- توزيع الحجوزات حسب الحالة (donut chart).
- الرحلات القادمة والتنبيهات النشطة.
- ملخص المدفوعات والمستحقات.
- أعلى 5 بائعين خلال آخر 30 يوم.
- توزيع السكن في مكة/المدينة/جدة.

### 9. التقارير والتحليلات (Reports & Analytics)

- **تقارير محاسبية**: ميزان مراجعة، أستاذ عام، قائمة الأرباح والخسائر، أعمار ديون الموردين.
- **تقارير تشغيلية**: تقرير الرحلات، قائمة المعتمرين، أداء البرامج.
- **تحليلات إدارية**: أداء المبيعات، أعلى العملاء، الربحية الشهرية، العمولات، المستحقات المتأخرة.
- **تصدير**: كل التقارير قابلة للتصدير Excel + PDF احترافي.

### 10. الأمان والصلاحيات (Security)

| العنصر | المعيار |
|---|---|
| الصلاحيات (Permissions) | عشرات الصلاحيات الدقيقة عبر Spatie Roles & Permissions |
| الأدوار (Roles) | مدير نظام / محاسب / موظف مبيعات / موظف عمليات / موظف خدمة عملاء — قابلة للتخصيص |
| سجل التدقيق (Audit Log) | كل تعديل/حذف على البيانات الحساسة (الحجوزات، القيود، الموظفين) يُسجَّل مع الـ user والوقت والتفاصيل |
| تأمين الجلسات | Session hardening + Email verification + Password reset |
| منع الاحتيال | حماية ضد الدفع الزائد، منع استرداد المبلغ مرتين، منع تكرار رقم الحجز |
| نسخ احتياطي | نظام Backup تلقائي مدمج |

### 11. التكاملات الخارجية (Integrations)

- **WhatsApp Cloud API** (الإشعارات والمتابعة).
- **منصة سفر (المسار)** — مزامنة بيانات المعتمرين والتأشيرات (mock layer جاهز).
- **منصة اعتماد العمرة** — مزامنة الحجوزات (mock layer جاهز).
- جاهز للربط مع: ميزة / فوري / إنستاباي / SMS gateway.
